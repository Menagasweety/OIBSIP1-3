# OIBSIP1-3:    TempConvert Pro 🌡️
A modern, responsive temperature converter built with HTML, CSS, and JavaScript featuring real-time formula updates, dual unit selection, and a premium UI design.

## 🚀 Features
- Convert between Celsius, Fahrenheit, and Kelvin
- Real-time formula display
- Clean and responsive UI
- Input validation with error handling
- Smooth animations

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
